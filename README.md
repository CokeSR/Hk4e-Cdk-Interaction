# 功能说明
## 根目录下
- 首页: DailySignIn.html
- 服主快捷功能：AccountAuth.html
## 在Index目录下
- CDK兑换：CdkExchange.php
- CDK添加：AddCdk.php
- CDK批量生成：AddCdkRandom.php
- CDK个人使用：AddCdkCommon.php
- 玩家GM后台：PlayerGmBackstage.php
- 服主授权平台1：PlayerAuthorization.php
- 服主授权平台2：PlayerBackstage.php
## 如何部署
- 宝塔建站或php建站即可
